# MathSystem
### 一. 為什麼我選C++？
> 1.  **C++是物件導向程式設計(Object Oriented Programing,OOP)，在傳統的程序式語言（如 C 語言）是以「專注於步驟、動作」的方式來寫程式；而 C++ 則引進了「物件」的概念，把資料（屬性）和操作資料的函式（行為）綑綁在一起，讓程式碼更貼近現實世界的運作方式。**
> 2. **如果是初學者，先學C++是一個正確的選擇！目的是想跟初學者說「這支C++程式設計不是可以考APCS，而是可以練思維能力！」**

### 二. C++四大核心的特性：
**1.封裝(Encapsulation)：封裝是指將資料和操作這些資料的方法包裝在同一個單元（即 class，類別）中，並隱藏內部的實作細節。**
> **好處： 外部程式碼不能隨便修改物件內部的私有資料，確保安全性。**

**2.繼承(Inheritance)：繼承允許一個新類別（衍生類別）繼承既有類別（基礎類別）的屬性和方法。**
> **好處： 減少重複程式碼，提高程式碼的重用性。**

**3.多型(Polymorphism)：多型指的是「同一個介面，多種實現方式」。讓不同的物件對同一個訊息做出不同的回應。**
> **好處： 增加程式的靈活性與擴充性。例如呼叫 animal.makeSound()，是狗就會叫「汪汪」，是貓就會叫「喵喵」。**

**4.抽象(Abstraction)：抽象是只呈現必要資訊，隱藏複雜的幕後邏輯。**
> **好處： 降低系統的複雜度，讓開發者專注於物件「該做什麼」，而不是「怎麼做」。**
